Rest Api для создания и управления задачами

Как запустить (для незнающих):
1)скачайте проект (zip или гит командой)
2)в корне проекта перейдите в myapp(cd myapp)
3)напишите composer install
4)перейдите в .env и настройте базу данных на свою 
5)ваша база данных должна быть запущена
6)сделайте миграцию - php artisan migrate
7)напишите php artisan serve
8) в другой вкладке в смд в папке frontend напишите php -S localhost:3000
9)перейдите в localhost:3000(http://localhost:3000/)
