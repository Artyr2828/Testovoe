# Testovoe
Тестовое Rest Api

1)для запуска скачайте зип файлом или перенесите командой гитом
2)Перейдите в .env и настройте Базу данных чтобы она работала у вас или оставьте SQLITE
3)скачайте все нужно - composer install
4) сделайте миграция в свою Базу данных - php artisan migrate
5) Запустите проект - php artisan serve
